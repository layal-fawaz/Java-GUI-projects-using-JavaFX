# 📚 Library Management System – JavaFX

The Library Management System is a desktop application developed using JavaFX, designed to streamline library operations such as user management, book tracking, and borrowing/returning processes. It follows Object-Oriented Programming (OOP) principles and the Model-View-Controller (MVC) architecture for better structure and maintainability. The interface is built with Scene Builder, ensuring a clean and user-friendly experience.

---

## ✨ Key Features

1. **🔐 User Authentication & Roles**  
   - Secure login using username and password.  
   - Role-based access control for Admin, Librarian, and Users.  
   - Each user is directed to a personalized dashboard based on their role.

2. **👥 User Management**  
   - Add, update, or delete users while maintaining their records in the database.  
   - Search for users by name or role and view a list of all registered members.

3. **📖 Book Management**  
   - Add books with title, author, edition, and category details.  
   - Edit or delete book records while keeping the database updated.  
   - Filter books by category for better organization.  
   - Search books by availability status (borrowed, pending, approved, rejected).

4. **🔄 Borrowing & Returning System**  
   - Users can request to borrow or return books.  
   - Librarians can approve or reject borrowing requests.  
   - Track borrowed books, due dates, and return status.  
   - Update book availability upon return.

5. **💾 Database & Data Storage**  
   - All data (users, books, transactions) is stored in a MySQL database.  
   - The system connects to the database using JDBC, ensuring real-time updates and data integrity.

---

## 🛠️ Technologies Used

- JavaFX for the graphical user interface.  
- Scene Builder for UI design.  
- JDBC for database connectivity.  
- MySQL for efficient data storage.  
- MVC (Model-View-Controller) for structured and scalable application design.

---

This system provides an efficient way for libraries to manage books and users, making borrowing and returning books organized, fast, and seamless.
